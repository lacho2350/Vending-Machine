PROJECT Vending Machine Test
Vending Machine that takes coins and retun treats and change.

Requirements
For development, you will need Node.js installed on your environment as well as the readline module.

Node is really easy to install & now include NPM. You should be able to run the following command after the installation procedure below.

$ node --version
v6.9.1(It should be fine with any later version )

$ npm --version
3.10.8

Installing:
node - https://docs.npmjs.com/downloading-and-installing-node-js-and-npm
readline- npm install readline.

Deployment
Go to the project folder opean console and type: "node console"
